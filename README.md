# SteadyDesk IT

Багатосторінковий сайт на чистому HTML, CSS, JavaScript і PHP. Без фреймворків та збирачів. SVG-іконки надає Lucide Icons 1.24.0, підключений у зафіксованій версії без package-файлів.

## Структура

- `index.html` та папки сторінок — готові HTML-файли.
- `assets/styles.css` — усі стилі сайту.
- `assets/app.js` — Lucide-іконки, мобільне меню, анімації, config-підстановка та AJAX-відправка форми.
- `config.js` — єдине джерело повторюваних даних: назва сайту й компанії, email, адреса, footer, внутрішні маршрути, навігаційні підписи, форма, collaboration-блок і cookie banner.
- `api/contact.php` — валідація і надсилання форми через стандартну PHP-функцію `mail()`.
- `images/` і `og.png` — зображення сайту.

## Локальний запуск

```bash
php -S localhost:8000
```

Відкрити `http://localhost:8000/`.

## Форма

На сервері має бути PHP 8+ і налаштована відправка пошти для функції `mail()`. У `config.js` потрібно вказати фактичну корпоративну адресу отримувача та адресу відправника з домену сайту. Об'єкт у цьому файлі потрібно залишати у JSON-сумісному форматі: без коментарів і зайвих ком після останніх значень.

## Деплой

Скопіювати вміст репозиторію в корінь PHP-сайту без жодної збірки або встановлення пакетів. Перед публікацією потрібно перевірити `contact.email`, `contact.website`, `forms.fromEmail` і маршрути в `navigation.routes`.

GitHub можна використовувати для зберігання коду, але GitHub Pages не виконує PHP. Для робочої форми потрібен PHP-хостинг із налаштованою поштою або сервер, що передає `mail()` у SMTP.
